🥗 Calorie Calculator Web Application

A responsive Calorie Calculator and Fitness Guide web application that helps users estimate daily calorie needs, track food intake, and learn about exercises and healthy diets. Built using HTML, CSS, JavaScript, and Bootstrap.

📌 Project Overview

This project allows users to:

Calculate their daily calorie requirements based on personal details

Track food intake and total calories consumed

Learn about calories burned through exercises

Understand healthy vs unhealthy diet choices

It is designed to promote health awareness and fitness planning through an easy-to-use interface.

✨ Features
🔢 Calorie Calculator

Inputs:

Age

Gender

Weight

Height (cm or feet/inches)

Activity level

Fitness goal (maintain, loss, gain)

Calculates estimated daily calorie needs

🍎 Food Calorie Tracker

Add food items and servings

Displays a running list of consumed food

Calculates total calories consumed

🏃 Exercise Information

Displays common exercises

Shows estimated calories burned (per 30 minutes)

Uses visual icons for better understanding

🥗 Diet Awareness

Explains healthy diet components

Highlights risks of unhealthy eating habits

🔍 Food Search (Navbar)

Search bar for food-related queries (frontend-ready)
